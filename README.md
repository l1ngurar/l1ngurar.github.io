# СкладРасклад
Результат двухдневных насильственных действий по отношению к DeepSeek с целью упростить еженедельную рутину в одной конторе. Да, я это навайбкодил, и что вы мне сделаете?

# Что это вообще такое?

Главный замысел - избавиться от постоянного бега между обычной клавиатурой и клавиатурой с эмодзи путём создания весьма удобного интерфейса, в котором уже есть все модели и цвета. Остаётся только записать количество, оставшееся на складе и нажать кнопочку, которая скопирует итоговый результат в нужном виде в буфер обмена, из которого, в свою очередь, это будет отправлено в рабочий чатик. Да. 

# ФуНкЦиОнАл

1. Автоматически генерирует отчёт в нужной нам форме, где каждая строка выглядит примерно так:

*12 128 - 1❤️ 2💙*

2. Сохраняет в кеше браузера последние 50 отчётов, с возможностью отредактировать, скопировать и удалить каждый 
3. Имеет адаптивную тёмную тему
4. Устанавливается как PWA-приложение
5. Умеет работать офлайн и брать актуальную версию "приложения" из этого репозитория
6. Коряво, но реализованы черновики - неоконченные отчёты, которые не потеряются, если закрыли сайт/приложение. При повторном открытии "приложения" возвращает Вас в редактор и помечает, что на данный момент мы доделываем незаконченный отчёт. Будет переработано, т.к. с главного экрана вернуться к черновику сейчас невозможно и меня это не устраивает

С огромной вероятностью чего-то нового здесь появляться не будет, разве что по мере поступлений новых моделей в ассортимент они отразятся и здесь.
